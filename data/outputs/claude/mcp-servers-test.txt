Based on the available functions, I have access to these MCP servers and their tools:

**Available MCP Servers:**

1. **ols-mcp** - Ontology Lookup Service
   - `search_all_ontologies` - Search across ontologies
   - `get_ontology_info` - Get ontology details
   - `get_terms_from_ontology` - Get terms from specific ontology

2. **landuse-mcp** - Land Use and Geographic Data
   - `get_land_cover` - Get land use data for coordinates
   - `get_soil_type` - Get FAO soil type for coordinates  
   - `get_landuse_dates` - Get available dates for land use data

3. **nmdc-mcp** - National Microbiome Data Collaborative
   - `get_collection_names` - List NMDC collections
   - `get_collection_stats` - Get collection statistics
   - `get_all_collection_ids` - Get document IDs from collections
   - `get_random_collection_ids` - Get random sample of IDs
   - `get_samples_in_elevation_range` - Filter by elevation
   - `get_samples_within_lat_lon_bounding_box` - Filter by coordinates
   - `get_samples_by_ecosystem` - Filter by ecosystem type
   - `get_entity_by_id` - Retrieve entity by ID
   - `get_entity_by_id_with_projection` - Retrieve with field projection
   - `get_entities_by_ids_with_projection` - Bulk retrieve with projection
   - `get_study_for_biosample` - Get study from biosample
   - `get_biosamples_for_study` - Get biosamples from study
   - `get_study_doi_details` - Get DOI details for study
   - `search_studies_by_doi_criteria` - Search studies by DOI

4. **artl-mcp** - Article and Literature Tools
   - `get_doi_metadata` - Get DOI metadata
   - `get_abstract_from_pubmed_id` - Get PubMed abstracts
   - `get_doi_fetcher_metadata` - Enhanced DOI metadata
   - `get_unpaywall_info` - Find open access versions
   - `get_full_text_from_doi` - Extract full text
   - `get_full_text_info` - Full text availability info
   - `get_text_from_pdf_url` - Extract text from PDFs
   - `clean_text` - Clean extracted text
   - `extract_pdf_text` - Standalone PDF extraction
   - `extract_doi_from_url` - Extract DOI from URLs
   - `doi_to_pmid` / `pmid_to_doi` - Convert between identifiers
   - `get_doi_text` / `get_pmid_text` / `get_pmcid_text` - Get full text
   - `get_pmid_from_pmcid` - Convert PMC to PMID
   - `get_full_text_from_bioc` - Get BioC format text
   - `search_pubmed_for_pmids` - Search PubMed
   - `search_papers_by_keyword` - Search scientific papers
   - `search_recent_papers` - Search recent publications

5. **weather-context-mcp** - Weather Data
   - `get_weather` - Get weather data for coordinates

All servers are loaded and functional.
