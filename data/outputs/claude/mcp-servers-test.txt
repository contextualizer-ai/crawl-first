Based on the MCP tools available in my function list, I have access to the following MCP servers and their tools:

**OLS-MCP** (Ontology Lookup Service):
- `search_all_ontologies` - Search across all ontologies
- `get_ontology_info` - Get detailed ontology information
- `get_terms_from_ontology` - Get classes/terms from specific ontologies

**Landuse-MCP**:
- `get_land_cover` - Get land use data for coordinates
- `get_soil_type` - Get FAO soil type for coordinates  
- `get_landuse_dates` - Get available dates for land use data

**NMDC-MCP** (National Microbiome Data Collaborative):
- Collection management (get_collection_names, get_collection_stats)
- ID retrieval (get_all_collection_ids, get_random_collection_ids)
- Spatial queries (get_samples_in_elevation_range, get_samples_within_lat_lon_bounding_box)
- Ecosystem queries (get_samples_by_ecosystem)
- Entity retrieval (get_entity_by_id, get_entity_by_id_with_projection, get_entities_by_ids_with_projection)
- Study/biosample relationships (get_study_for_biosample, get_biosamples_for_study)
- DOI management (get_study_doi_details, search_studies_by_doi_criteria)

**ARTL-MCP** (Article/Literature tools):
- DOI metadata (get_doi_metadata, get_doi_fetcher_metadata)
- PubMed integration (get_abstract_from_pubmed_id, doi_to_pmid, pmid_to_doi)
- Full text retrieval (get_full_text_from_doi, get_pmid_text, get_pmcid_text)
- PDF processing (get_text_from_pdf_url, extract_pdf_text)
- Literature search (search_pubmed_for_pmids, search_papers_by_keyword, search_recent_papers)
- Unpaywall integration (get_unpaywall_info)

**Weather-Context-MCP**:
- `get_weather` - Get hourly/daily weather data for coordinates

The MCP resource listing shows no specific resources are currently loaded, but all the above tools are available for use.
